# UI Overview

## Screens
- **Login**: simple auth entry point.
- **Onboarding**: capture role and basic profile info.
- **Client Dashboard**: show bookings, documents, and messages.
- **CA Dashboard**: show client queue and service requests.
- **Document Center**: manage uploads and status.
- **Booking Flow**: create and confirm services.

## Components
- **TopNav**: app title + user actions.
- **BottomNav**: quick navigation for mobile.
- **InfoCard**: reusable summary card.
