# ca-platform-mvp

Monorepo scaffold for the CA platform MVP with a lightweight Express backend and a Vite + React frontend.

## Quick start

### Backend
```
cd backend
cp .env.example .env
npm install
npm run dev
```

### Frontend
```
cd frontend
npm install
npm run dev
```
